# 基因编辑效率分析 Bash 脚本
CRISPResso2 analysis of high-throughput sequencing data
该脚本用于复现Evaluation and minimization of Cas9-independent off-target DNA editing by cytosine base editors文章中fig.2c，使用CRISPResso2分析高通量测序数据，统计基因编辑效率。

## 功能
- 分析原始数据
- 计算编辑效率
- 输出结果到指定文件夹
